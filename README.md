## JD直播助手PLUS

京东建场+投屏+AI话术

### 1. 京东建场

- 选择商品文件
  - 默认是2个商品文件
  - 可以新增商品文件
  - 可以调整商品的放置顺序
  - 显示选中商品文件的有效数据
- 选择标题
  - 支持AI生成多个标题
  - 支持设置标题，一行一个
- 设置数量
  - 单品文件数量
  - 流量品文件数量
  - 总数量（打满多少）
- 浏览器数据
  - 获取到浏览器登录cookie
  - 获取到登录的账号昵称
  - 根据选中的浏览器去执行新建直播间


### 2.AI话术

- 按照添加的商品去生成AI话术
- 缓存商品数据+对应的AI话术


### 3.投屏

- 创建图片窗口，展示商品图片
- 创建倒计时窗口，显示倒计时
- 开始直播前，等待1分钟
- 每个商品，都要按照等待休息时间
- 每个商品，要执行调用点击讲解，结束讲解
